Neo4j for Ruby

What is Neo4j.rb?

Neo4j.rb (neo4j gem) is a Ruby wrapper for the Neo4j graph database. It is a “Drop in” replacement for Rails Active Record.

Top Features

  • Object Model interface for Nodes and Relationships including:
    • Properties
    • Indexes / Constraints
    • Callbacks
    • Validation
    • Assocations
  • Query building / chaining interface similar to arel
  • Transactions
  • Migrations
